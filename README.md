## 📦 Requirements

```
Python 3.8+
aiohttp
colorama
aiohttp_socks (optional)
```

Install dependencies:

```bash
pip install aiohttp colorama aiohttp_socks
```

---

## 🚀 Usage

```bash
python vf77.py
```

Follow the CLI menu and select the command you want to run.

---

## 🔧 Proxy Format

Create a file called **proxies.txt**

```
http://user:pass@ip:port
http://ip:port
socks5://user:pass@ip:port
socks5://ip:port
socks4://ip:port
```
